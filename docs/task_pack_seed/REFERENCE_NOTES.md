# Reference Notes

These references informed the architecture choices. Verify them again before implementation if versions change.

- OpenAI Agents SDK: Agents are applications that can plan, call tools, collaborate, and keep state; use Agents SDK when your app owns orchestration, tool execution, approvals, and state.
- OpenAI Agents SDK Python docs: Agents support tools, handoffs, guardrails, structured outputs, and tracing.
- IBKR TWS API: TCP Socket Protocol API based on TWS or IB Gateway, used to retrieve and send data.
- IBKR paper trading: lets users test strategies in simulated environment without risking capital.
- Alpaca paper trading: paper-only account can use Alpaca API in paper environment.
- OpenBB Platform: can run local FastAPI REST API and integrate financial datasets.
- Freqtrade: crypto trading bot with backtesting and dry-run.
- Stripe Agent Toolkit: use sandbox and evaluations for agentic workflows; use restricted API keys for security.
